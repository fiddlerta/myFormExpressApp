# Simple ordering form, Express App
Available endpoints: 
- '/' - renders main page;
- '/users' - returns list of customers;
- '/success' - returns success page;

Open Cypress: node_modules/.bin/cypress open
Run app: npm run start
Run unit tests: npm run unit-tests
